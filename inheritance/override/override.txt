{Override}
{Defintion}
- Name, Parm, must match parent method.  
- Compiler chooses method based on actual object type. 

{Covariant}
- Covariant: The return type is allowed to be of a subClass of parent class. 

{Special occasions}
Super keyword: can use parent method.
Final methods: can not be overridden. 
Static methods: can not be overridden.
